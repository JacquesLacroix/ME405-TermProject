Detailed Description

This class implements a proportional gain controller for a DC motor with quadrature encoder.

Create a DC motor controller object with proportional gain from quadrature encoder feedback.

Parameters
encoderThe encoder object used to measure position
motorThe motor driver used to send signals to the motor
setpointThe target position for the motor to turn to
kpThe proportional gain coefficient for the controller to use
kiThe integral gain coefficient for the controller to use
kdThe derivative gain coefficient for the controller to use
ticksThe number of encoder ticks corresponding to 1 full rotation
gearRatioThe gear speed reduction ratio, allowing for angles to be set on the output shaft (Output Gear Teeth/Motor Pinion Teeth). A positive ratio should be used for odd numbers of gears, and a negative ratio should be used for even numbers of gears to ensure the output shaft rotates in the desired direction. The ratio should be negated if the motor is mounted upside-down relative to the output shaft.
